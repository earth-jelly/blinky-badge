# blinky-badge
This is a blinky board with an original graphic of Heidi the raccoon (and her braincells working hard to spark an 'Aha moment') on it.
<img width="1723" height="892" alt="image" src="https://github.com/user-attachments/assets/8b229080-8be2-4496-b4ab-70e80568929e" />

## Why?
When I first started this project, I envisioned this blinky board to be used as a light up badge for my (former) FRC team. However, as I was getting everything set up, I realized that I wanted this project to be something that I can be proud of and actually like, so I changed the overall theme to what I like, which is Heidi. 

I ultimately decided on the Aha! moment Heidi as my design because the nature of the blinky board (the fact that it lights up) reminded me of an lightbulb and an realization moment. This, as you can tell in the final product, translated directly to the silkscreen art on the board.

## So, what did I learn?
As a person who have tried making a PCB before, making the schematics and routing the components weren't anything new. However, importing footprints from online library, ordering the PCB, and creating a BOM were something that was completely new to me and that I had to learn (albeit, it took a little too long to do).
